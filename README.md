##  Ответы на контрольные вопросы  

### 1. Что делает тема и что делает плагин?  
Тема отвечает за внешний вид сайта, включая дизайн и оформление.  
Плагин добавляет дополнительную функциональность, например формы, редакторы и другие возможности.

### 2. Почему не теряется контент при смене темы?  
Контент хранится в базе данных, а тема отвечает только за его отображение.  
Поэтому при смене темы данные сайта сохраняются.


### 3. Как изменить внешний вид сайта без редактирования кода?  
Внешний вид можно изменить с помощью установки тем, а также через настройки в разделе «Внешний вид → Настроить».
